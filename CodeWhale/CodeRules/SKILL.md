# CodeRules — 编码规则速查

覆盖 250+ 条可操作的工程编码约束，按「编码流程 → 代码风格 → 安全合规 → 架构模块 → 分布式运维 → 产业专项」表格化组织。每条规则标注 P0/P1/P2 优先级，适合快速查阅和日常代码审查。附 Git 规范、CI/CD 门禁、脚本工程化等实战条目。

## When to use

- 生成任何面向生产环境的代码
- 修改/重构现有代码时确保不改坏既有逻辑
- 设计 API、数据库 Schema、分布式系统
- 编写测试用例、CI/CD 脚本、Dockerfile

## Core Principles

```
安全/正确性 > 可观测性 > 可读性 > 性能 > 极简性
```

低优先级让步时须注释豁免理由。

## 13. 产业专项扩展（场景按需激活）

### 13.1 金融/支付

账务数据只增不改（红字冲销）、交易原子性保障、日终日间对账强制、金额定点数禁止浮点、交易超时原路退回、敏感交易二次鉴权、灾备双活、报文防篡改。

### 13.2 医疗/HIS

病历数据写后不可修改（仅追加备注）、医疗数据访问全审计可追溯、诊疗流程状态硬校验、医疗设备通信可靠优先弱网重试、合规留存病历时长。

### 13.3 工控/IoT

控制逻辑执行时间可预期（禁用随机/动态加载）、IO 异常进安全稳态（停机/保位/报警）、固件升级签名验签防篡改、离线数据缓存联网补发。

### 13.4 车载/车规

严格栈大小限制（高频静态内存）、功能安全模块故障自检双路冗余、ECU 固定时序启动、意外断电状态自动落盘。

### 13.5 云原生/K8s

容器前台运行禁后台、CPU/内存/PID 硬配额、根文件系统只读、镜像最小化禁编译工具、liveness/readiness 探针、网络策略最小开放、preStop 优雅终止。

### 13.6 客户端/嵌入式

端侧内存上限管控、弱网适配三态区分、本地数据加密存储、日志分级脱敏定期清理、UI 主线程禁耗时操作、嵌入式静态内存优先。

### 13.7 脚本工程化

严格模式开启、变量入参校验、高危命令二次确认、日志与退出码规范、跨平台兼容。

### 13.8 政企/涉密

内外网物理隔离、文档密级绑定、等保审计日志不可篡改、个人信息最小采集、数据出境单独审批、账号口令复杂度强制、涉密数据分级标识。

### 13.9 高并发容灾

缓存穿透/击穿/雪崩三级防护、锁粒度最小化+超时、流量削峰队列缓冲、多实例负载分片、熔断隔离渐进恢复、冷热数据分离、降级应急开关标准化、生产禁热更、批量操作前置评估、配置变更灰度发布、测试数据禁入生产。

### 13.10 团队协作

目录按领域拆分（层级≤5）、代码合并前跑 Lint/UT/静态检查、语义化版本、配置文件逐条注释分离、废弃接口标注下线时间、前后端统一错误码、接口版本生命周期管理。
